# 🛸 Nuxt Rick and Morty App
![site](https://github.com/katianne23/Rick-and-Morty---Projeto-Nuxtjs/blob/main/public/images/preview.png)

> 💻 Projeto de estudo desenvolvido com **Nuxt.js**, consumindo a **API do Rick and Morty**, com foco em rotas, parâmetros, query strings, meta tags, layouts e implementação de um layout baseado no **Figma**.

Este projeto tem caráter educacional e faz parte do meu aprendizado em Nuxt.



[Clique aqui para acessar](rick-and-morty-projeto-nuxtjs.vercel.app)


## 🎨 Design

- Layout baseado em um protótipo do **Figma**
- Interface responsiva
- Componentização seguindo boas práticas

> ⚠️ Os botões de **favoritos** e **modo claro/escuro** estão presentes na interface, porém ainda **não possuem implementação funcional**.


## 🛠 Tecnologias Utilizadas:
- [Nuxt.js](https://nuxt.com/)
- [Vue](https://vuejs.org/)
- [typeScript](https://www.typescriptlang.org/)
- [tailwind](https://tailwindcss.com/)
- [NuxtUi](https://ui.nuxt.com/)
- [Api](https://rickandmortyapi.com)


## 📌 Funcionalidades Implementadas

### 🌐 Rotas
- Rotas automáticas
- Rotas dinâmicas com parâmetros (ex: `/character/[id]`)

### 🔎 Consumo de API
- Listagem de personagens
- Página de detalhes
- Uso de parâmetros e query strings

### 🧱 Layouts
- Layout base reutilizável
- Separação clara entre layout e páginas

### 🧠 SEO
- Meta tags dinâmicas por página
- Uso do `head()` do Nuxt

---

## ❌ Funcionalidades **não implementadas**

- ⭐ Listagem de personagens favoritos
- 🌗 Alternância entre modo claro e escuro
- Persistência de estado (localStorage / Pinia / Vuex)

Essas funcionalidades fazem parte do **roadmap do projeto**.

---

## ▶️ Como Executar o Projeto

1. Clone o repositório:
```
git clone https://github.com/katianne23/Rick-and-Morty---Projeto-Nuxtjs.git

Acesse a pasta:
cd rick-and-morty-project

Instale as dependências:
npm install

Execute o projeto:
npm run dev

Acesse:
http://localhost:3000
```
